# Daniel Hed - Portfolio

A personal portfolio website showcasing my work, skills, and projects as a full-stack developer.

## Tech Stack

- **Frontend:** React, Vite, Material-UI
- **Animations:** Framer Motion, custom CSS animations
- **3D Elements:** Three.js / React Three Fiber
- **Styling:** Material-UI, custom CSS

## Features

- Interactive UI with scroll-based animations and parallax effects
- Project gallery with detailed project pages
- Interactive console for FAQ-style questions (press ? or / to open)
- Responsive design for desktop and mobile
- Custom cursor and visual effects
